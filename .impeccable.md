## Design Context
### Users
ReadEra users who want to visualize backup citations from book readings and export them easily on web.

### Use Cases
- Import ReadEra backup files.
- Visualize books that contain quotes.
- Visualize and organize citations.
- Filter citations by color.

### Brand Personality
Simple, smooth, practical.

### Aesthetic Direction
Minimal modern tool UI.

### Constraints
Dark mode only.

### Accessibility
No formal WCAG target provided; preserve baseline accessibility best practices (semantic structure, keyboard support, visible focus states, sufficient contrast where possible).

### Design Principles
- Prioritize fast, frictionless workflows for importing, browsing, and exporting citations.
- Keep layouts minimal and task-focused, with clear hierarchy and low visual noise.
- Use smooth, predictable interactions that support practical daily use.
- Avoid outdated or generic AI-style visual patterns; maintain a modern, intentional interface.
- Ensure dark-mode readability and baseline accessibility in every core flow.
